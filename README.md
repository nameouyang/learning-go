# learning-go
### 

#### go 语言的基础项目
使用 gin + gorm 开发

项目结构

```
learning-go/:
    ├─biz/        # 业务逻辑中间层
    ├─build/      # 项目构建目录
    ├─cmd/        # 
    ├─conf/       # 配置文件
    ├─controller/ # 控制器
    ├─lib/        # 基础服务
    ├─middleware/ # 中间件
    ├─models/     # 模型
    ├─router/     # 路由目录
    ├─service/    # 服务层
    ├─go.mod      # 包管理文件
    ├─go.sum      # 依赖包版本哈希文件
    └─main.go     # 运行入口main文件
```
